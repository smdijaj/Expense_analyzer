# Expense Analyzer

A Python-based expense analysis project using CSV, SQL, Pandas, NumPy, and Matplotlib.

## Features

- Read CSV expense data
- Store in SQLite database
- Query expenses by category
- Generate category summaries
- Visualize spending charts
- Export reports
- Filter expenses

## Tech Stack

- Python
- Pandas
- NumPy
- SQLite
- Matplotlib
